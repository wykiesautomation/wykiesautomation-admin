# Wykies Admin (GitHub Pages)
Static admin frontend hosted on GitHub Pages under the subdomain `admin.wykiesautomation.co.za`.

## Deploy
1. Create public repo `wykies-admin` and push these files.
2. Enable GitHub Pages (Settings → Pages → Branch: main).
3. Custom domain: `admin.wykiesautomation.co.za` (CNAME included).
4. Replace `REPLACE_WITH_OAUTH_CLIENT_ID` and `REPLACE_WITH_DEPLOY_ID`.
