# Wykies Automation – Google Apps Script Backend

1) Paste `Code.gs` and `2.gs` into https://script.google.com project.
2) Set Script Properties: SHEET_ID, SECRET, ADMIN_EMAIL, ADMIN_PASS_SHA256.
3) Deploy → Web app (Execute as: Me, Who has access: Anyone). Use the /exec URL in web/config.json if you change deployments.
